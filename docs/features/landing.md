# Landing Page Specifications: India Employment Hub

## 1. Visual Strategy
**Goal:** Premium, modern, AI-first aesthetic inspired by Jobright.ai.
- **Color Palette:** Slate-50 background, Indigo/Sky-600 primary accents, soft gray borders.
- **Typography:** Modern Sans-serif (Inter/Outfit), tight tracking for headings.
- **Motion:** Subtle staggered entrance animations for hero elements.

## 2. Hero Section
- **Headline:** Focus on "AI-First Matching" and "End of Swiping".
- **Visual:** A dynamic "Vision" component showcasing AI match rings or real-time activity logs.
- **CTA:** Sticky search bar with "AI-Powered" badge.

## 3. The "Active" System Section
- **Interactive Explainer:** A visual timeline or toggle showing the difference between "Active" and "Passive" status.
- **Copy:** Explain the 4-day responsiveness rule for both employers and candidates.

## 4. Problem & Solution
- **Split-Pane Design:** Showcase the "Old Way" vs "IEH Way".
- **Focus:** No clutter, AI summarization, direct connections.

## 5. Feature Highlights
- **Cards Bento Grid:**
    - AI JD Generator (For Employers)
    - Match Rings (For Candidates)
    - Resume Analysis
    - Interview Prep AI
