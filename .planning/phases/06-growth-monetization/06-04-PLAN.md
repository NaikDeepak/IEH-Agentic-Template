# Plan: 06-04 - Brownie Points Ledger & Atomic Transactions

## Goal
Implement a robust, atomic ledger for tracking "Brownie Points".

## Proposed Changes

### 1. Service: Ledger Service
- Create `src/features/growth/services/ledgerService.ts`.
- `addPoints(uid, amount, type, metadata)`:
  - Uses `runTransaction`.
  - Increments `users/{uid}/browniePoints`.
  - Creates a new doc in `ledger/` collection.

### 2. UI: Points Balance
- Create a reusable `PointsBadge` component.
- Display current balance in the header/dashboard.

## Verification Plan
1. Manually trigger `addPoints`.
2. Verify balance updates and ledger entry is created.
3. Test with negative amounts (redemptions).
